# **App Name**: Saeed - A Lasting Legacy

## Core Features:

- Quran Streaming: Stream Quran recitations from various readers, dynamically constructing URLs for each Surah.
- Athkar Display: Display various Athkar (morning, evening, after prayer, etc.) with clear text, repetition counters, and subtle animations.
- Personalized Dua: Dedicated section for Dua (supplication), including a custom prayer for the user's father.
- Daily Reminders: Configure reminders for Surah Al-Kahf, morning/evening Athkar, daily verses, fasting, etc.
- Intent Display: Display a static text explaining the app is a Sadaqah Jariyah (ongoing charity) for the user's father.
- AI-Powered Verse Suggestion: Suggest verses relevant to the user's current situation using generative AI tool.

## Style Guidelines:

- Primary color: Soft, muted green (#A7D1AB) to evoke tranquility and growth.
- Background color: Very light desaturated green (#F0F4EF), providing a calm backdrop.
- Accent color: Muted blue (#95B8D1) to complement the green with a touch of serenity.
- Body and headline font: 'IBM Plex Arabic', a beautiful font for clear Arabic text.
- Use soft, rounded icons, providing a consistent feel across the app.
- Clean, modern layout optimized for Arabic reading, ensuring ease of use.
- Subtle fade and slide animations to enhance user experience without being distracting.
